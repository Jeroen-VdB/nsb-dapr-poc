# nsb-dapr-poc
Proof of concept to validate bidirectional messaging between a service using NServiceBus and a service using Dapr.

# Prerequisites
1. VS Code
1. dotnet 8
1. Docker Desktop or Podman
1. Dapr
1. Azure Service Bus Namespace Standard or Premium
